
### Project Phoenix: The Conversational DevOps Assistant
An intelligent, multi-agent AI assistant that unifies and automates developer workflows across GitHub, Jira, and your IDE through a single, conversational interface.

---
### Key Features
* Conversational Interface: No need to remember complex commands. Just talk to Phoenix like you would to a team member.
* Multi-Agent System: Powered by specialized agents for Git, Jira, Development, and Research.
* Multi-Step Plan Execution: Can understand and execute chained commands from a single query.
* Interactive Code Editor: A unique feature that opens an in-app dialog for direct code editing.
* Stateful Memory: Remembers the context of your conversation for seamless multi-turn interactions.

---
### Technology Stack
* Backend: Python, CrewAI, MCP servers, LLM
* Frontend: Streamlit
* AI Service: Azure OpenAI / OpenAI
* Core Integrations: GitHub API, Jira API
